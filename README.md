# 🔢 BaseWizard – Multi Base Number Converter

A sleek, real-time number system converter built with **HTML**, **CSS**, and **JavaScript**. Converts between **Decimal**, **Binary**, **Octal**, and **Hexadecimal** instantly as you type. Designed with a modern UI and smooth UX — fast, responsive, and perfect for geeks, learners, and devs.

---

## 🚀 Features

- 🔁 **Real-time conversions** across all four number systems
- 💡 **Smart input detection** with validation
- 🌈 **Color-coded fields** for each base
- 🌓 **Dark-themed modern UI** for hacker vibes
- 📱 **Fully responsive** — looks slick on mobile too
- 📋 **One-click copy** buttons for each output
- 🧠 **Tooltips** explaining each number system
- 🔄 Optional **base swap feature** (coming soon!)
